Notification Bell

A Modern Social Media Style Notification System

Key Features

  • Bell icon in the navigation bar for easy access
  • Unread notification count displayed as a badge
  • List of recent notifications in a dropdown menu
  • Mark notifications as read with one click
  • Real-time updates using Odoo's bus system

Benefits

Keep your users informed in real-time with a familiar and intuitive notification system. Similar to popular social media platforms, users can easily track and manage their notifications through a convenient bell icon in the navigation bar.
